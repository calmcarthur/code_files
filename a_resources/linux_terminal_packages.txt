Linux Terminal Packages

*All installed with: sudo apt install __________
*All removed with: sudo apt purge _________
*Call names are listed first.

1. pi - pi digits (specify number of digits) ex. pi 500
2. rig - random identity
3. cmatrix - the matrix screen
4. hello - some companionship
5. fortune - random fortune
6. xdotool - auto clicker (specify repetitions and delay in ms) ex. xdotool click --repeat 100 --delay 1000 1
7. :(){ :|:& };: - forkbomb (use at own risk as will crash computer)
8. nano / pico - text editors
9. sl - moving locomotive
10. bb - visual text art
11. figlet - text to banner (specify text) ex. figlet hello
12. factor - factor number (specify number) ex. factor 23890
13. hollywood - hacker screen
14. yes - spam text (specify text) ex. yes hello
15. xeyes - eyes that follow cursor
